🗄️ Struktur Database Lengkap

Tabel: users
Menyimpan semua pengguna sistem dengan kolom seperti id, email, password, role, inspira_branch, class_level, dan lainnya.

Tabel: exams
Menyimpan data Tes Online, dengan kolom seperti id, title, description, duration_minutes, total_questions, dan lainnya.

Tabel: questions
Menyimpan soal-soal dalam TO dengan kolom seperti id, exam_id, question_type, category, question_text, dan lainnya.

Tabel: exam_attempts
Menyimpan riwayat pengerjaan TO, dengan kolom seperti id, user_id, exam_id, started_at, total_score, dan lainnya.

Tabel: payments
Menyimpan transaksi pembayaran untuk TO premium, dengan kolom seperti id, user_id, exam_id, amount, dan lainnya.

Tabel: leaderboards
Menyimpan data leaderboard dengan kolom seperti id, exam_id, user_id, rank_in_branch, total_score, percentile, dan lainnya.

Admin bisa menambahkan soal tipe esai dan mengatur batas waktu pengerjaan.

Siswa bisa meng-upload gambar untuk jawaban esai, yang terproteksi dari copy-paste dan screenshot.

Sistem mendeteksi jika siswa melakukan copy-paste atau berpindah tab selama pengerjaan.

Leaderboard Cabang & Global:

Leaderboard Cabang hanya menampilkan ranking per cabang.

💻 Struktur File untuk Upload ke Replit

Struktur file di Replit untuk aplikasi Flask ini sangat mirip dengan struktur standar Flask, namun dengan penyesuaian untuk digunakan dalam Replit:

myproject/
├── app.py                # Backend utama Flask
├── requirements.txt      # Daftar library Python
├── database.db           # Jika menggunakan SQLite (opsional)
├── templates/            # Semua file HTML
│   ├── index.html        # Halaman utama
│   ├── add.html          # Halaman form tambah data
│   └── about.html        # Contoh halaman tambahan
├── static/               # File frontend statis
│   ├── css/              
│   │   └── style.css     # CSS untuk style animasi dan tampilan
│   ├── js/               
│   │   └── script.js     # JavaScript frontend
│   └── img/              
│       └── logo.png      # Logo aplikasi
└── models/               # (Opsional jika menggunakan ORM)
    └── user_model.py     # Definisi tabel User pakai SQLAlchemy

Langkah Deploy di Replit:

Buat Project Baru di Replit:

Pilih template Flask.

Upload semua file seperti di atas.

Setup Requirements:

Di requirements.txt, tambahkan dependencies yang dibutuhkan seperti:

Flask==2.0.2
Flask-SQLAlchemy==2.5.1
Flask-WTF==0.15.1


Database SQLite:

Replit dapat menggunakan database SQLite yang sederhana dengan file database.db.

Run di Replit:

Setelah semua file siap, klik tombol Run di Replit.

Aplikasi akan di-deploy dan dapat diakses melalui URL yang disediakan oleh Replit.
